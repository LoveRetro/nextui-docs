# Device Support

NextUI is built for specific TrimUI handhelds. Install it only on a device listed here unless maintainers publish instructions for another device.

## Officially supported devices

- TrimUI Brick
- TrimUI Smart Pro
- TrimUI Smart Pro S
- TrimUI Brick Hammer, which is functionally the same as the TrimUI Brick

## Unsupported and experimental devices

If a device is not listed above, do not assume a NextUI release supports it. Experimental ports can exist before they are ready for normal users, and those builds need their own maintainer instructions.

For other handhelds, check [MinUI](https://github.com/shauninman/MinUI).

## Paks on different devices

Paks are optional add-ons maintained by NextUI contributors and community members. A Pak must support your device before it appears or works correctly.

If a Pak is missing from Pak Store, launches back to the menu, or behaves differently on Smart Pro S, check the Pak's own compatibility notes and update the Pak before filing a firmware bug.

See [Paks](../paks.md) and [Pak Store](../pak-store.md) for installation and troubleshooting.
