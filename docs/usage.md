## What you need

- A supported device: Trimui Brick, Trimui Smart Pro, or Trimui Smart Pro S.
- A microSD card from a reputable vendor.
- A computer that can extract `.zip` files and copy files to the SD card.
- The latest NextUI release archive ending in `-base.zip` or `-all.zip` from the [GitHub releases page]({{ urls.github }}/releases).
- Your own ROM and BIOS files. NextUI does not provide copyrighted games or BIOS files.

## Supported devices

| Device | Status | Platform | Bootstrap folder | Notes |
|---|---|---|---|---|
| Trimui Brick | Supported | `tg5040` | `trimui` | Main NextUI target device. |
| Trimui Smart Pro | Supported | `tg5040` | `trimui` | Shares platform and Paks with Brick. |
| Trimui Smart Pro S | Supported | `tg5050` | `trimui` | Some Paks may need explicit Smart Pro S support. |
| Trimui Brick Hammer | Supported (same as Brick) | `tg5040` | `trimui` | Functionally identical to the Brick. Same SD card and Paks. |

For more details, see [Device Support](reference/device-support.md).
