# Customizing

NextUI supports a range of customization options for artwork, overlays, shaders, themes, LEDs, boot logos, and more.

- [Artwork](customizing/game-artwork.md) — add box art and media images for your games.
- [Overlays](customizing/overlays.md) — add game-specific overlay borders and bezels.
- [Shaders](shaders/index.md) — apply visual effects like CRT scanlines and LCD grids.
- [LEDs, Power, and Clock](customizing/power-leds-clock.md) — LED controls, sleep behavior, battery tracking, and RTC settings.
- [Custom Boot Logo](customizing/custom-boot-logo.md) — replace the boot splash screen.
- [Emulator Icons](customizing/emulator-icons.md) — add icons for systems, tools, and collections.
- [Quick Menu Theme](customizing/quick-menu-theme.md) — customize quick menu backgrounds.
- [Collections](customizing/custom-categories.md) — organize games into custom categories.
