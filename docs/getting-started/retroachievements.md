# RetroAchievements

NextUI added RetroAchievements support in v6.9.0.

RetroAchievements require:

- a RetroAchievements account;
- Wi-Fi;
- a supported emulator/core;
- a game file that can be identified by hash.

## Logging in

1. Connect the device to Wi-Fi.
2. Open Settings.
3. Open the RetroAchievements login option.
4. Enter your username and password with the on-screen keyboard.
5. Save the settings.

If you need to change your login later, use the Settings menu again.

## Supported features

NextUI v6.9.0 added:

- login through Settings;
- achievement unlock notifications with badge images;
- progress tracking;
- leaderboard support;
- game hash identification, including CHD disc image support;
- automatic badge caching and asynchronous badge downloading;
- rich presence support.

The same release added in-game toast notifications for save/load state operations, screenshots, volume/brightness changes, and achievement unlocks.

## Hardcore mode

Standard RetroAchievements are supported. Hardcore mode is not currently available in NextUI.

## If achievements do not appear

Check:

1. Wi-Fi is connected.
2. You are logged in through Settings.
3. The game is supported by RetroAchievements.
4. The game file matches a recognized hash.
5. You are using a supported core.
6. Try an unmodified No-Intro-style ROM if your current ROM is not detected.
7. For disc games, prefer formats documented as supported by NextUI's RetroAchievements integration.

## Do I need RetroArch?

No. NextUI's RetroAchievements integration is part of NextUI. You do not need to launch RetroArch separately for normal NextUI RetroAchievements support.
